# Sonidos del Bingo

Coloca aquí los archivos MP3 para los eventos de juego:

- `line-win.mp3` - Se reproduce cuando un cartón gana LÍNEA
- `bingo-win.mp3` - Se reproduce cuando un cartón gana BINGO (cartón lleno)

## Formato recomendado
- MP3, 128kbps o superior
- Duración: 2-5 segundos
- Volumen normalizado
